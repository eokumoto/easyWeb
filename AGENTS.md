# EasyWeb

EasyWeb is a browser designed for older adults who want a safer and simpler internet experience.

## Mission

Build a calm, accessible browsing experience that helps seniors confidently complete everyday online tasks while allowing trusted family members to provide remote assistance.

## MVP

- Senior homepage with large bookmarks
- Browser page
- AI "Need Help?" assistant
- Helper dashboard
- Shared bookmarks between helper and senior

## Design Principles

- Prioritize simplicity over adding features.
- Use large buttons and readable typography.
- Avoid clutter.
- Every screen should have a clear primary action.
- Minimize the number of decisions users must make.

## Accessibility

- Large touch targets
- High contrast
- Plain language
- No hidden gestures
- Responsive on desktop and mobile

## Engineering

- Use TypeScript.
- Keep components reusable.
- Do not refactor unrelated files.
- Make the smallest change necessary.
- Explain implementation plans before coding.
- After coding, summarize which files changed.

## AI Safety

Never claim to complete actions like submitting forms or making payments.

The AI should guide users, not impersonate websites or perform sensitive actions automatically.
